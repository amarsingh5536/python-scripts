#In this tutorial you can make local server website live using Ngrok.

Ngrok The best option is to use a tool to expose your current local server. Exactly! People all around the world will be able to see whatâ€™s happening on your localhost.

settings & steps expose a Django local server website to the internet.

1. Download and install ngrok in your oprating system: https://ngrok.com/download

2.  create a ngrok URL on any port(8000) where people can access our server
by command : ngrok http 8000

example : Web Interface                  http://127.0.0.1:4040                                                            
		 Forwarding                      http://4c52-2409-4051.ngrok.io -> http://localhost:8000                            
		 Forwarding                      https://4c52-2409-4051.ngrok.io -> http://localhost


3. copy Forwarding ngrok url "https://4c52-2409-4051.ngrok.io"


4. Paste forwarding ngrok copied url in ALLOWED_HOSTS = ['https://4c52-2409-4051.ngrok.io'] or ['*']


5. Run python manage.py runserver / your app on port 8000


* Now your website is live, you can access your local website on internet on url 'https://4c52-2409-4051.ngrok.io'.
